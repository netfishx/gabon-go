# 云服务约束：仅 S3 与阿里云 CDN 豁免

本版尽可能不依赖云厂商托管服务，仅有两项豁免：对象存储走 S3 协议（AWS S3 或任意 S3 兼容实现），内容分发被限定使用阿里云 CDN（回源 S3）。其余能力一律自托管或进程内实现——转码用 ffmpeg（ADR-0003）、队列与调度在进程内（DB 表 + cron）、密钥走环境变量、无云 MQ/云监控/托管数据库依赖。目的是保持部署自主性与本地可闭环验证，避免旧版那种"AWS 全家桶深绑定"（S3 + MediaConvert + 明文云凭证散落配置）的形态。

## Consequences

- 未来引入任何云托管服务（含监控、告警、MQ、Serverless）都应先挑战本 ADR，而非默认可用
- 防盗链如需启用，用阿里云 CDN 自带的 URL 鉴权，而不是自建签名网关；当前维持"不签名"现状

# 平行实验品定位：不与旧系统兼容

本项目不接管旧 gabon 的现网流量与数据，是与旧系统平行存在的实验品。因此 API 形状、数据库 schema、密码哈希算法（旧版为自制 salt+sha256，本版用 argon2id）、错误码体系均可自由重设计，不承担任何逐字节兼容义务。验收基准不是"与旧系统对拍一致"，而是 `docs/feature-checklist.md` 的全量功能清单逐项达成。

## Consequences

- 旧库数据不迁移；旧客户端无法直连本服务
- 未来若要接管现网，需要另行设计数据迁移与 API 适配层——届时再决策，本仓库不预留

## 修正（2026-07-03）

定位从"纯实验品"升级为**候选替代品**：若验证可行，本系统将替换旧版接管现网。随之调整两点：

- 质量基准按生产标准执行（如对外 ID 防枚举），不再以"实验品"为由降档
- 数据迁移从"不会发生"改为"可能发生"：schema 不做关闭迁移之门的选择（主键保持 bigint，旧库 id 可原值迁入）；兼容义务仍不承担——API 形状自由设计，调用方配合修改
